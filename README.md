# <a id="english"></a>English

Config for Neovim (lua)

## Requirements:
  - Lua
  - Neovim >= 0.8.0 (0.9.5 or higher recommended)
  - git (optional)
  - Nerd font (optional)

## Installation
Enter this:
```sh
mkdir -p ~/.config/nvim && git clone https://github.com/alexei-pav/neovim-cfg.git ~/.config/nvim
```

## Update:
```sh
cd ~/.config/nvim && git pull
```

## Shortcuts:
Leader = Space

### Normal mode
| Shortcut | Action |
| -------- | ------- |
| Ctrl+Q  | Exit    |
| Leader+W | Save file |
